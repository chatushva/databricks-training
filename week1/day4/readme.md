# What I Learned

Through this SQL practice project, I learned the fundamentals and intermediate concepts of SQL using the `Employees` table in MySQL.

## Key Learnings

* Learned how to create tables and insert data into a database.
* Practiced retrieving data using `SELECT` statements.
* Understood how to filter records using the `WHERE` clause.
* Learned to use aggregate functions like `SUM()`, `AVG()`, `COUNT()`, `MAX()`, and `MIN()`.
* Gained knowledge of grouping data using `GROUP BY`.
* Learned how to filter grouped data using `HAVING`.
* Practiced sorting data using `ORDER BY`.
* Learned to retrieve limited records using `LIMIT`.
* Understood the use of `DISTINCT` to remove duplicate values.
* Practiced comparison operators such as `>`, `<`, `>=`, `<=`, and `<>`.
* Learned logical operators like `AND`, `OR`, and `NOT`.
* Understood how to use `IN` and `NOT IN` for multiple value conditions.
* Practiced range filtering using `BETWEEN`.
* Learned pattern matching using the `LIKE` operator with `%` wildcard.
* Improved problem-solving skills by writing different SQL queries.

## Outcome

This project improved my confidence in writing SQL queries and helped me build a strong foundation in database management and data retrieval concepts.
